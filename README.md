## Publish on GitHub Pages
1) Upload ALL files to repo root.

2) Settings → Pages → Source: Deploy from branch; Branch `main`; Folder `/ (root)` → Save.

3) Visit https://USERNAME.github.io/REPO/

## Add real images
Place JPG/PNG in `assets/img/`. Keep filenames same as JSON to auto-swap; or update `data/artworks.json` to your desired names.
