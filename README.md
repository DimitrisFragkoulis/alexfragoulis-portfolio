# Alex Fragoulis — Site v7 (Aglaia-style landing + full portfolio)

## What you get
- `index.html` → ultra-minimal landing ("Coming Soon" + Instagram), inspired by the public layout of aglaiazorba.com home.

- `printmaking.html`, `illustration.html`, `other.html` → full portfolio pages, data-driven.

- `assets/js/site.js` → smart fallback image loader.

- `data/artworks.json` → edit/add to grow the gallery.

## Publish on GitHub Pages
1) Upload ALL files to your repo root.

2) Settings → Pages → Source: Deploy from branch; Branch: `main`; Folder: `/ (root)` → Save.

3) Visit https://USERNAME.github.io/REPO/

## Add real images
Put JPG/PNG files in `assets/img/` using any names; update `data/artworks.json` accordingly OR keep placeholder base names to auto-swap.
