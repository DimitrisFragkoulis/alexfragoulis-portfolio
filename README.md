# Alex Fragoulis — Portfolio v4 (GitHub Pages optimized)

### How to publish on GitHub Pages
1) Upload ALL the files in this folder to your repo root (not inside a subfolder).
2) Repo → Settings → Pages → Source: **Deploy from branch**; Branch: **main**, Folder: **/** (root).
3) Wait 1–2 minutes → Your site is live at: https://USERNAME.github.io/REPO-NAME/

### Add new images later
- Put files in `assets/img/` with any filename.
- Add an entry in `data/artworks.json` pointing to `assets/img/yourfile.jpg`.
- Commit → Pages updates automatically.
